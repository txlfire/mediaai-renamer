# 2026-07-09 M9-2 登录和前端权限

## 本次范围

- 实现前端登录、管理员初始化、当前用户展示和退出。
- 接入全局 401 会话失效处理。
- 按用户权限禁用首批已接后端守卫的高风险操作入口。

## 实现内容

- 新增认证 API client：`bootstrapAdmin`、`login`、`logout`、`fetchCurrentUser`。
- 新增 auth store，保存当前用户、权限列表和登录状态。
- 新增 `/login` 页面，支持登录和初始化首个管理员。
- 路由守卫要求非公开页面必须登录。
- 侧边栏显示当前用户，退出按钮调用真实退出接口。
- API 401 响应会清理本地 token 并跳转登录页。
- 媒体源、扫描任务、系统设置和重命名页面按用户权限禁用对应写操作入口。
