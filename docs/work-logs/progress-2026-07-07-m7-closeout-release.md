- TMDB / AI 匹配来源补齐 `parent_folder_title`。
- 单条 AI、批量 AI、TMDB→AI 后备链路统一透传匹配来源。
- 上级文件夹标题命中敏感词时，同样阻止外部提交并记录拦截结果。

### 3. 增量扫描稳定性

- 扫描任务处理阶段引入 savepoint，失败时回滚本次写入。
- 历史索引在失败任务下保持不变。
- 未变化文件的增量扫描不会重复生成新的预览识别结果。
- 历史小文件不会因阈值变更被误判为重新失效。
- 扫描失败兜底文案统一补充中文前缀，前端直接可读。
