# M7-3.5B 批量 AI 匹配前端接入

日期：2026-07-06

版本：v0.7.8

## 本次范围

- 在重命名页接入后端批量 AI 解析接口 `POST /api/rename-previews/ai-parse/batch`。
- 新增“AI 匹配”和“全部AI匹配”入口，分别支持选中条目和当前列表中未匹配/低置信条目的批量 AI 解析。
- 操作前弹出确认提示，明确说明会提交文件名、路径、解析标题、上级文件夹标题等必要信息，并会先经过敏感词和外部提交保护。
- 批量 AI 解析结果只作为候选缓存，不自动回填、不执行真实重命名。
- 行内 AI 操作优先打开已缓存的批量结果，用户仍需手动选择候选后回填。
- 操作结果弹窗补充成功、失败、拦截/跳过数量和 Token 用量日志。

## 未纳入本次范围

- TMDB 匹配失败后自动串联 AI 二次匹配。
- AI Provider 一键切换管理界面。
- 批量 AI 结果的跨刷新持久化。

## 验证

- `npm.cmd run frontend:test`
- `npm.cmd run frontend:build`
