# v0.6.9 正式版发布记录

日期：2026-07-03

## 发布范围

- 完成 M6 AI 智能解析正式版发布准备，版本号由 `0.6.8` 升级为 `0.6.9`。
- 修复重命名文件名解析中技术参数组导致的标题误截断问题，例如 `(BDRip 1920 FLAC)` 不再被识别为年份。
- 优化重命名页底部统计入口，改为统一“统计”气泡展示，减少对分页控件的挤压。
- 统一表格分页、全屏按钮、空状态和上下间距样式，提升扫描结果、重命名、媒体源等表格页面的一致性。
- 补充 M7 增量扫描阶段设计、开发计划和验收清单，包含所属文件夹辅助解析、手动排除到待处理列表、最小扫描文件阈值回归验证等内容。

## 发布边界

- 本次正式版不包含 M7 的增量扫描实现。
- M7 相关内容仅作为下一阶段设计和计划，不写入已完成功能说明。
- 批量 AI 匹配、TMDB 与 AI 串联匹配仍按 M7 计划推进。

## 验证记录

发布前需完成以下验证：

- `npm.cmd run backend:test`
- `npm.cmd run frontend:test`
- `npm.cmd run frontend:build`
- `npm.cmd run check:encoding`
- `git diff --check`
- `npm.cmd run release:package`
