- 修复系统设置页 `v4Token` 表单字段缺失导致的前端构建失败。

- TMDB 匹配支持 V4 只读访问令牌优先调用，V4 失败时自动降级到 V3 API Key。
- 命名预览高置信匹配会记录实际生效通道，如 `TMDB (V4)` 或 `TMDB (V3)`。
- 补充 V4 优先、V3 降级、V4/V3 都失败、令牌脱敏和预览通道回写测试。
