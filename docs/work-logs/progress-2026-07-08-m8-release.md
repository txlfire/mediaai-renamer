# 2026-07-08 M8 发布准备

版本：`0.8.0`

## 结论

- M8 命名规则高级配置二期已完成主要开发、文档同步和发布前验证。
- 本次发布范围限定在 M8 已落地能力，不包含 M9 权限、回滚、审计和任务治理实现。

## 本次发布内容

### 1. 命名规则管理

- 命名模板支持电影和剧集模板版本号、更新时间。
- 支持模板导入、导出和结构校验。
- 规则测试返回生成结果、字段命中情况和格式告警。
- 差异预览可对比当前生效模板和待保存模板输出。

### 2. 预览链路联动

- `rename_previews` 记录生成时使用的模板类型、模板版本和模板更新时间快照。
- 历史预览可展示当前模板、旧模板或版本未知状态。
- 模板更新不会自动改写历史目标文件名。
- 手动编辑过的目标文件名仍受重新生成确认保护。

### 3. 前端展示

- 命名预览列表对旧模板和版本未知记录显示轻量标签。
- 命名预览详情展示模板版本状态。
- 系统设置规则测试结果展示字段命中和格式告警。

### 4. 文档同步

- 更新 README 当前能力和文档索引。
- 更新 M8 开发计划、验收清单和设计手册。
- 新增 M8 用户手册。
- 更新完整用户手册到 M1-M8 当前已实现能力。

## 发布说明建议

- Release 标题：`v0.8.0`
