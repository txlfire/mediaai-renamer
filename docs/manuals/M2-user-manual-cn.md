# MediaAI Renamer M2 用户手册

版本：v0.2.0 预览版
阶段：M2 标准命名与预览
日期：2026-06-24

## 1. 当前可以做什么

M2 可以基于已经扫描出的媒体文件生成标准命名预览。

你可以查看原文件名、解析结果、建议目标文件名和预览状态，也可以手动编辑目标文件名。

M2 不会修改真实文件名。

## 2. 进入命名预览

在左侧菜单点击“命名预览”。

页面包含：

- 生成预览按钮。
- 刷新按钮。
- 媒体源筛选。
- 扫描任务筛选。
- 状态筛选。
- 类型筛选。
- 关键字搜索。
- 预览表格。
- 编辑弹窗。

## 3. 生成预览

操作步骤：

1. 先在 M1 页面完成媒体源扫描。
2. 进入“命名预览”页面。
3. 可选择媒体源或扫描任务作为筛选条件。
4. 点击“生成预览”。
5. 等待列表刷新。

如果不选择筛选条件，系统会对当前数据库中的全部媒体文件生成预览。

## 4. 预览状态

状态说明：

- 可使用：系统识别为电影或剧集，并生成了标准目标文件名。
- 需检查：系统无法完全判断类型、年份或季集信息，需要人工确认。
- 已编辑：用户已经手动修改过目标文件名。

## 5. 命名规则

电影默认格式：

```text
标题.年份.扩展名
```

示例：

```text
The.Matrix.1999.mkv
```

剧集默认格式：

```text
标题.S季号E集号.扩展名
```

示例：

```text
Show.Name.S02E03.mp4
```

## 6. 编辑目标文件名

在预览表格中点击“编辑”。

输入新的目标文件名后点击“保存”。

注意：

- 只能编辑文件名，不能输入目录路径。
- 如果没有输入扩展名，系统会自动补回原扩展名。
- 编辑只保存到预览记录，不会修改真实文件。

## 7. 当前限制

M2 不支持：

- 执行真实重命名。
- 检测重名冲突。
- 选择目标目录。
- 外部元数据匹配。
- AI 解析。
- 批量确认执行。

这些能力会进入后续阶段。
