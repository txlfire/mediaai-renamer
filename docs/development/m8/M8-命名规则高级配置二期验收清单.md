# M8 命名规则高级配置二期验收清单

## 1. 配置兼容

- [ ] 旧命名模板配置可直接读取，不因新增版本字段报错。
- [ ] 旧配置保存后会补齐版本信息和更新时间。
- [ ] 导入旧格式模板时可给出兼容处理或明确错误提示。

## 2. 模板导入导出

- [ ] 可导出当前电影模板和剧集模板配置。
- [ ] 导入模板时会校验 JSON 结构、字段定义和格式参数。
- [ ] 非法模板不会覆盖当前生效配置。

## 3. 规则测试与差异预览

- [ ] 用户可输入样例并查看模板生成结果。
- [ ] 测试结果会展示字段命中情况和格式警告。
- [ ] 保存前可对比当前模板结果与待保存模板结果差异。
- [ ] 差异预览不会改写正式命名预览记录。

## 4. 双语命名

- [ ] 电影模板可配置双语字段组合方式。
- [ ] 剧集模板可配置双语字段组合方式。
- [ ] 缺少中文标题或英文标题时能自动跳过空字段和多余分隔符。

## 5. 历史预览联动

- [ ] 新生成的命名预览会记录模板版本号。
- [ ] 模板更新后，历史预览可提示基于旧模板生成。
- [ ] 对手动编辑过的目标文件名，重生成前仍有明确确认提示。

## 6. 回归

- [ ] M2 命名预览基础流程正常。
- [ ] M3 dry-run 和安全重命名正常。
- [ ] M4 TMDB / IMDb 回填后命名构建正常。
- [ ] M6 AI 候选回填后命名构建正常。
- [ ] M7 标题识别模式和手动排除正常。
