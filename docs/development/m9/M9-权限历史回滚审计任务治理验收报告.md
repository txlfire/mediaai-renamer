# M9 权限、历史、回滚、审计和任务治理验收报告

版本：`0.9.1`
日期：2026-07-09

## 1. 验收结论

M9 已达到正式发布条件。`v0.9.0` 为正式发布版本；`0.9.1` 作为 M9+ 维护版补齐任务治理归档入口。版本范围包括本地认证、直接用户权限、敏感接口守卫、审计日志、操作运行日志、重命名回滚、任务治理和手动归档能力。

## 2. 已完成范围

- 数据库 schema 升级到 `17`，新增用户、会话、审计、回滚、操作日志和任务归档相关表。
- 后端实现本地管理员初始化、登录、退出、当前用户、改密、默认管理员密码提示和隐藏重置开关。
- 前端实现登录页、登录态、401 / 403 处理和按权限展示或禁用高风险入口。
- 敏感写接口接入权限守卫，包括设置保存、媒体源写操作、扫描创建、真实重命名、回滚、外部提交和任务治理。
- 审计日志覆盖登录、用户维护、配置保存、媒体源变更、外部提交覆盖、真实重命名和回滚。
- 操作运行日志覆盖扫描、重命名 dry-run、真实重命名和回滚，支持查询、增量轮询、清理、导出和脱敏。
- 重命名回滚支持创建计划、dry-run 校验和执行，不覆盖已有文件。
- 任务治理页面支持查看扫描任务、重命名批次和回滚计划，并可跳转目标页面、查看运行日志、手动归档和恢复任务。
- README、设计文档、开发计划、验收清单、用户手册和工作日志已同步。

## 3. 延期范围

- 复杂自动归档策略延期到后续增强；手动归档和恢复已在 M9+ `0.9.1` 完成。
- 企业级多租户、LDAP、OIDC、OAuth、双因素认证不属于 M9 范围。
- WebDAV、FTP、SFTP、S3 等远程协议仍属于后续阶段。

## 5. 发布说明

`v0.9.0` 是 M9 正式发布版本，重点是治理能力，不改变 M1-M8 的核心扫描、匹配、预览和安全重命名使用路径。`0.9.1` 是 M9+ 维护版，补齐任务治理手动归档入口。
